Spacing

Spacing is our tempo knob. Turn it with restraint and the interface reads like a well-spaced zine--airy where it needs to breathe, tight where it needs to click.

Spacing Scale

Pick a single cadence (like 4 or 8) and treat it like the tick marks on a ruler. Every gap should land on that beat so the layout never drifts off rhythm.

Applied spacing in components

Buttons, cards, and lists should share the same internal padding so they feel cut from the same sheet. Use larger steps between sections and smaller steps inside components to keep hierarchy crisp.

Spacing tokens

Name spacing like labeled dials: xs, sm, md, lg--or s1, s2, s3. Tokens keep the system consistent even when the layout gets remixed.
