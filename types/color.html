FOUNDATION_MODULE_05

Colour

Colour is the phosphor glow and paper stock of the system. Keep it nostalgic and disciplined: warm neutrals, deep inks, and a few high-signal accents.

Core Palette

Start with dependable bases (ink, paper, graphite) and add a couple of retro brights (amber, cyan, signal red). The palette should feel like it came from a screen-print kit.

Color Roles

Assign roles--background, surface, text, border, accent, success, warning, danger--so colors behave like labeled channels. Roles beat random hex codes every time.

Usage Combinations

Pair high-contrast text with quiet surfaces, then reserve accents for primary actions and key states. If everything glows, nothing glows.

Do / Don't

Do: keep contrast strong and accents rare. Don't: stack multiple saturated colors in one component or rely on color alone to communicate status.
